# DDTank 4.1 Documentation

This repository contains the source code and documentation for DDTank version 4.1, an open-source server-client ecosystem.

## Project Documentation Index

- **[Technical Overview](architect/tech_overview.md)**: Detailed information about project components, technologies used, and development conventions.
- **[Architecture & Workflow](architect/architecture.md)**: Visual representation of component relationships and high-level gameplay workflow.
- **[Setup Guide](architect/setup.md)**: Comprehensive instructions for setting up the server, database, web environment, and desktop client.

## Quick Start Summary

The project consists of a C# multi-component server, an ASP.NET Request handler, a PHP user portal, and an Electron-based desktop client wrapping the original Flash game.

For detailed setup instructions, please refer to the **[Setup Guide](architect/setup.md)**.
